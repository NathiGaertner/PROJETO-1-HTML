Conceito de Client

Neste tema, se debruça em nos explicar que Client trata-se dos nossos computadores, que é diferente do servidor. Passa pelo tópico de navegador onde fica claro que se trata da ferramenta que usamos para navegar na internet. E também podemos ver a aplicação web que são soluções para que possamos navegar na internet sem instalação, e nesta ordem passa pela parte de dispositivos móveis e seu promissor mercado.

Servidores

Aqui podemos ver mais detalhadamente como funciona a comunicação entre Client x Servidor e a internet no meio deles. Existem alguns tipos de servidores, com finalidades diversas.

Curso 2 - Introdução ao HTML na Prática

Ferramentas utilizadas

Para iniciar devemos baixar o sofwere Visual Studio Code e também precisamos de uma navegador. Descobrimos o inspesor de elementos e logo vamos para a estrutura básica do HTML. Dai em diante conseguimos ver melhor a carinha e como de fato a coisa toda funciona. Adentramos nas tags, textos e sua estrutura, lista ordenadas e não ordenadas e como fazer uma página que o usuário possa navegar na inetent atraves da nossa criação dos links

Devemos anexar este projeto no repositório do site GitHub para que tenhamos um portfolio nesde o ínicio.

Por fim, o professor nos relembra de algumas tags e mostra outras que ficou faltando.
